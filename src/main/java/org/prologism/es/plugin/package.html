Provides the classes necessary to register the plugin into an elasticsearch node.

This example plugin {@link org.elasticsearch.plugins.AbstractPlugin} contains 3 main component, each one is a a factory for instanciate 3 kind of class :

  • Analyzer {@link org.elasticsearch.index.analysis.AbstractIndexAnalyzerProvider} : run during the tokenization step of ES (after indexation step)
  • Scipt {@link org.elasticsearch.script.AbstractSearchScript}: called during a request of elasticserach, it could be called in a facelet or in simpple request
  • River {@link org.elasticsearch.river.River}: a loop in a thread which can make some opeartion on the index, or listen a other datasource
